Desarrollo Web

El desarrollo web es el proceso de crear y mantener un sitio web que sea funcional en internet.El desarrollo web implica el uso de diferentes lenguajes de programación, según el modelo y la parte de la página que corresponda. Un desarrollador web o un programador es alguien que toma un diseño web y lo convierte en un sitio web mediante la escritura de código.El desarrollo web busca que una web tenga una apariencia impecable, un funcionamiento rápido y un buen desempeño para permitir la mejor experiencia de usuarios.
